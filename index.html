The Plan

  • Get git installed on your laptops
  • Register with github.com
  • Get a local copy of the remote project.
  • Go through the workflow of making a change and getting that change merged into the original project.

git

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Getting Git

From the git home page, click on the Downloads link.

Grab the version that is appropriate for your OS.

Don't know git at all?

Github is the most popular public host for git repositories.
